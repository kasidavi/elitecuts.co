## 📝 Customization Guide

### Update Phone Number
Edit `index.html` line 86 and change the phone number:
```html
<p><a href="tel:+1234567890">+1 (234) 567-8900</a></p>
```

### Add Gallery Images
Replace the placeholder gallery items in `index.html` with actual images:
```html
<div class="gallery-item">
    <img src="path/to/your/image.jpg" alt="Cut description">
</div>
```

### Update Business Info
Edit the contact section in `index.html` to match your details.

### Change Colors
Edit `styles.css` to customize:
- Gold color: `#ffd700`
- Dark color: `#1a1a1a`
